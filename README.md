该项目为WeeeeChat客户端，服务器仅保存用户基本信息和用户进入的房间号，所有聊天记录均保存在客户端本地。

服务端地址：https://github.com/37166121/WeeeeChatServer.git

socket默认端口为9999

API默认端口为58574
